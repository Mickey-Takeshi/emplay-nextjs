- ホームページ制作: 「制作費用」「相場」「予算」の組み合わせが多く、種類別の総額と見積条件を先に提示する。

## 編集方針

1. 既存記事の情報量は維持し、検索者が最初に必要とする結論・比較表・手順を冒頭へ置く。
2. 検索結果のタイトルと本文の主題を一致させる。
3. 固定料金や製品仕様は公式情報で確認し、契約条件で変わるものは断定しない。
4. 根拠のない全国一律の採用単価などは掲載せず、自社実績を計算・分解できる内容にする。
5. 自己リンク、重複見出し、存在しないダウンロード案内を削除する。

## 効果確認

公開後28日を目安に、同じページ単位でクリック、表示回数、CTR、平均掲載順位を比較する。判断時はクエリ構成の変化も確認し、表示回数が増えたため平均順位が一時的に下がるケースと、既存クエリの順位低下を分けて扱う。

順位上昇は保証できないため、1回の更新で結論を出さず、次回は次の条件で再調整する。

- 表示回数増・CTR低下: タイトルとディスクリプションを再検討
- 8〜15位で停滞: 不足見出し、一次情報、内部リンクを追加
- クリック増・問い合わせなし: 記事内CTAとサービス適合性を確認
- 順位・表示回数とも低下: 検索意図のずれ、重複記事、事実更新を再監査
